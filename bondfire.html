Technologies Used

We needed to build a light and reactive mobile application so decided between React Native and Flutter, choosing the former due to its large library support.

With the backend, it needed to compliment our React Native JS code so after research, Meteor JS was chosen as our backend Node JS framework and MongoDB as our database.

What I learned

Building a full-stack mobile application has a lot of intrciaties and it is important to carefully review th architeure of how you will build each system.

For example, how will you handle the Redux and state management of all your data or how will data pass through your React components.



Challenges

This was my deep dive into React Native and a lot of time was spending understanding the modules and packages. So these are issues that I am familiar with going forward with my development.

Understanding different styling dimensions and adhering to a style guide that was collaboratively made with design and engineering team was essential.
